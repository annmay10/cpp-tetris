### Game

#### Keyboard

| Key            | Action                                                      |
|----------      |-------------------------------------------------------------|
| `right arrow`  | Move tetromino to the right                                           |
| `left arrow`            | Move tetromino to the left                                            |
| `down arrow`      | Move tetromino down                                                        |
| `up arrow`            | Rotate tetromino                                         |
